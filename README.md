# NRATrainz Project
NRATrainz adalah website katalog addons Trainz Simulator berbasis Laravel, Filament, TailwindCSS, dan PostgreSQL.

## Fitur Utama
- *Coming Soon*: Fitur utama akan segera hadir, termasuk katalog addons yang lengkap, sistem manajemen pengguna, dan banyak lagi.

## Teknologi yang Digunakan
- **Backend**: Laravel
- **Frontend**: Filament, TailwindCSS
- **Database**: PostgreSQL
